tkmap.js

Documentation for version 3.0.0

Examples

TkMap({lat:42.01048,lng:-87.6652,domid:'map',init:true});
Map = new TkMap({domid:'map',lat:42.01048,lng:-87.6652,zoom:12});
Map.setMapOptions({zoomControl:false});
Map.setResponsive();
Map.setCustomStyles({styles:'grey minlabels hybrid'});
Map.initMap();

See a working example

Constructor

TkMap ({
  domid : {string},
  init : {boolean},
  lat : {float},
  lng : {float},
  zoom : {int}
});

Arguments

The order in which arguments are defined in the constructor is meaningless. However domid, lat, and lng are required.

domid

string required The id value of the HTML DOM object into which the map will render. This is usually a <div> tag. Example of the required HTML: 

<div id="map"></div>

init

boolean optional If set to 'true', the map will render as part of the instantiation of the class. If set to 'false', the map will not render until the .display method is called.

lat

float required The map center latitude upon initial render. In decimal degrees.

lng

float required The map center longitude upon initial render. In decimal degrees.

zoom

integer optional The map zoom level upon initial render. If this argument is not passed, the initial zoom level is set to 15.

Properties

.Map

This is the google.maps.Map object. Any google.maps.Map methods from the Google Maps Javascript API should be directed at this property.

Methods

.initMap()

Displays the map in the specified HTML DOM object. If the constructor's init argument is not set, or set to 'false', this .initMap() method must be called.

.setCustomStyles()

Example

Map.setCustomStyles({styles:'road grey minlabels'});

The map is styled with a series of defined styles. Separate styles with a space.

  • road: A road map.
  • satellite: A satellite map.
  • hybrid: A hybrid map. A satellite map with roads and labels.
  • terrain: A terrain.
  • grey: A greyscale colored map.
  • minlabels: Only road, neighborhood, city, state, and country labels shown.

.setResponsive()

The initial map zoom level will vary based on the map DOM element's pixel width. Below 481 pixels, the zoom level is default zoom - 1. Above 980 pixels, the zoom level is default zoom + 1.

.setMapOptions()

Example

Map.setMapOptions({zoomControl:false});

Extend the TkMap.MapOptions property with more options from google.map.MapOptions object. If you pass an option already defined, this method will overwrite the previous value.

Options set by default are:

center : new google.maps.LatLng(<lat & lng from constructor>),
mapTypeControl : false,
mapTypeId : google.maps.MapTypeId.ROADMAP,
panControl : false,
streetViewControl : false,
styles : [],
zoom: <zoom from constructor or 15>,
zoomControl : true

tkmap.js

Documentation for version 2.1.0

Constructor

TkMap (
  <draw> {boolean},
  <dom_id> {string},
  <mapCenterLatitude> {float},
  <mapCenterLongitutde> {float},
  <zoomLevel> {int:optional},
  <responsiveZoom> {boolean:optional},
  <mapStyleOptions> {string:optional};
);

Examples

TkMap(true,'map',42.01048,-87.6652,15,true,'road minlabels grey');
themap = new TkMap(false,'map',42.01048,-87.6652);
themap.display();

Arguments

<draw>

boolean If set to 'true', the map will render as part of the instantiation of the class. If set to 'false', the map will not render until the .display method is called.

<dom_id>

string The id value of the HTML DOM object into which the map will render. This is usually a <div> tag. Example of the required HTML: 

<div id="map"></div>

<mapCenterLatitude>

float The map center latitude upon initial render. In decimal degrees.

<mapCenterLongitude>

float The map center longitude upon initial render. In decimal degrees.

<zoomLevel>

integer optional The map zoom level upon initial render. If this argument is not passed, the initial zoom level is set to 15.

<responsiveZoom>

boolean optional If set to 'true', the initial map zoom level will vary based on the browser width (innerWidth). Below 481 pixels, the zoom level is <zoomLevel> - 1. Above 980 pixels, If this argument is not passed, the zoom level is <zoomLevel> + 1. If this argument is not passed, the default value is 'false'.

<mapStyleOptions>

string optional The map is styled with a series of defined styles.

  • road: A road map.
  • satellite: A satellite map.
  • hybrid: A hybrid map. A satellite map with roads and labels.
  • terrain: A terrain map.
  • grey: A greyscale map.
  • minlabels: Only road, neighborhood, city, state, and country labels shown.

Properties

.map

This is the google.maps.Map object. Any google.maps.Map methods from the Google Maps Javascript API should be directed at this property.

Methods

.display()

Displays the map in the specified HTML DOM object. If the constructor's <draw> argument is set to 'false', this .display() method must be called.

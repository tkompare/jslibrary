tksocrataview.js

Documentation for version 1.0.0

Examples

var Licenses = TkSocrataView({
  viewid:'b7mg-yyns',
  domain:'data.cityofchicago.org'
});

See a working example

Learn about Socrata

Constructor

TkSocrataView ({
  viewid : {string},
  domain : {string}
});

Arguments

The order in which arguments are defined in the constructor is meaningless. However viewid and domain are required.

viewid

string required The id value of the Socrata data view. This is usually four alphanumeric characters, followed by a dash (-), followed by four more alphanumenric characters. For example: b7mg-yyns

domain

string required The domain name of the Socrata powered data wherehouse. For example: data.cityofchicago.org

Properties

None public

Methods

.getData()

Get the object that holds the data from the Socrata View. Data is addressed by row number and then column name. For example: LicenseData[0].doing_business_as_name

Each column name is the SODA.meta.view.columns[i].fieldName passed in the JSON string response from Socrata. As viewed on the Socrata-powered data web site, the 'doing_business_as_name' column is labeled 'DOING BUSINESS AS NAME'. The fieldName is always the diplayed column name in all lower case, and underscores in place of spaces.

Warning: this method will not return data until the data is finished downloading from the data source. 

var LicenseData = Licenses.getData();
console.log(LicenseData[0].doing_business_as_name);

tkmapfusionlayer.js

Documentation for version 2.0.0

Constructor

TkMapFusionTableLayer (
  <draw> {boolean},
  <mapObject> {object},
  <fusionTableId> {string},
  <geoColumn> {string},
  <icon> {string:optional}
);

Examples

TkMapFusionLayer(true,Map.map,'3028961','Location');
var TreeTrimLayer = new TkMapFusionLayer(false,Map.map,'3028961','Location','grn_blank');
TreeTrimLayer.showLayer();

Arguments

<draw>

boolean If set to 'true', the Fusion Tables map layer will render as part of the instantiation of the class. If set to 'false', it will not render until the .showLayer method is called.

<mapObject>

object The google.maps.Map object onto which this Fusion Tables data layer is will display. If you are using TkMap base map object, this would be the 'TkMap.map' property.

<fusionTableId>

string The 'Numeric ID' of the Google Fusion Table.

<geoColumn>

string The Fusion Table column that holds the geographic point, line, or polygon information. In decimal degrees.

<icon>

string optional The name of the Google Fusion Table icon to use. Point map only.

Methods

.showLayer()

Displays the map layer in the specified google.maps.Map object. If the constructor's <draw> argument is set to 'false', this .showLayer() method must be called. If no arguments are passed, all data rows are mapped.

Arguments

<type>
string optional If set to 'search', this method will perform a "CONTAINS IGNORING CASE" search on the Fusion Tables data.

<string>
string optional The string value on which this method will search the Fusion Tables data.

<keyColumn>
string optional The name of the Fusion Tables column to search.

<icon>
string optional The name of the Google Fusion Table icon to use. Point map only. If not passed, and this is a point layer, it will default to the TkMapFusionLayer.icon property value, if defined.

Example

TkMapFusionLayer.showLayer('search','Open','Status','red_blank');

.hideLayer()

Hides the map layer in the specified google.maps.Map object.

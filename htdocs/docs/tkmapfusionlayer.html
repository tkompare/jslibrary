tkmapfusionlayer.js

Documentation for version 3.0.0

Examples

var MapLayer = new TkMapFusionLayer({
  map:themap.Map,
  tableid:'3028961',
  geo:'Location'
});
var MapLayer = new TkMapFusionLayer({
  map:themap.Map,
  tableid:'3028961',
  geo:'Location',
  where:"Status CONTAINS IGNORING CASE 'Open'"
});

See a working example

Constructor

TkMapFusionLayer ({
  geo : {string},
  icon : {string},
  map : {object},
  tableid : {int},
  where : {string}
});

Arguments

The order in which arguments are defined in the constructor is meaningless. However geo, map, and tableid are required.

geo

string required The name of the Google Fusion Tables column that holds the geographic location information for the row of data. The geographic information must be in decimal degrees.

icon

string optional The name of the Google Fusion Table icon to use. Point map only. The value must be one from the 'iconName' list of supported Google icons.

map

object required The google.maps.Map object onto which this Fusion Tables data layer is will display. If you are using TkMap base map object, this would be the 'TkMap.Map' property.

tableid

integer required The 'Numeric ID' of the Google Fusion Table to place on the map.

where

string optional The WHERE clause for the Google Fusion Table query. If this argument is not defined, all data rows are mapped. Follow the specification found in the Google Fusion Tables documentation. The syntax is generally: <column_name> <operator> <value>

Methods

.showLayer()

Example

MapLayer.showLayer({icon:'red_blank',where:"Status = 'Open'"});

Displays the Google Fusion Table map layer in the specified 'google.maps.Map' object.

Arguments

icon
string optional The name of the Google Fusion Table icon to use. Point map only. If not passed, and this is a point layer, it will default to the TkMapFusionLayer.Icon property value, if defined. The value must be one from the 'iconName' list of supported Google icons.

where
string optional The WHERE clause for the Google Fusion Table query. If this argument is not defined, it will default to the TkMapFusionLayer.Where property value, if defined. Follow the specification found in the Google Fusion Tables documentation. The syntax is generally: <column_name> <operator> <value>

.hideLayer()

Hides the map layer in the specified google.maps.Map object.

tksocrataview.js Demo

This example displays Chicago's 49th Ward 2012 business licenses.

Include required JS libraries, just above the end </body> tag.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="js/tksocrataview.1.0.0.js"></script>
<script src="js/tksocrataviewcontroller.js"></script>

Here is this example's controller Javascript tksocrataviewcontroller.js. 

$(document).ready(function() {
  BusinessLicenses = new TkSocrataView
  ({
    tableid : 'b7mg-yyns',
    domain : 'data.cityofchicago.org'
  });
});
